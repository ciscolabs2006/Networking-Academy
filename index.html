CCNA 1

Introduction To Networks

This module covers the fundamentals of networking, including the OSI and TCP/IP models, Ethernet, and IP addressing (IPv4/IPv6). Students learn to configure basic network devices like routers and switches, perform simple troubleshooting, and understand basic security concepts. This course lays the groundwork for advanced networking topics in later CCNA modules.

  • Course Duration: 5 Weeks
  • Cost Ksh. 5,000

CCNA 2

Switching and Routing Essentials

CCNA 2 focuses on configuring and troubleshooting routers, switches, and wireless networks in small to medium-sized enterprises. Key topics include VLANs, static and dynamic routing, and network security. Students also explore wireless network fundamentals and gain practical skills in managing and securing network devices. This course builds on the foundational knowledge from CCNA 1, preparing students for more complex networking tasks.

  • Course Duration: 5 Weeks
  • Cost: Ksh. 6,500

CCNA 3

Enterprise Networking, Network Security and Automation

CCNA 3 delves into advanced networking concepts, including scalable network design, advanced routing protocols (like OSPF and EIGRP), and network security solutions. Students also explore the automation of network tasks using tools like Python and understand how virtualization and SDN (Software-Defined Networking) shape modern enterprise networks. This final course prepares students for real-world networking environments and the CCNA certification exam.

  • Course Duration: 5 Weeks
  • Cost: Ksh. 6,500

LFS101

Linux Essentials

Linux Essential Courswork provides a beginner-friendly introduction to the Linux operating system. The course covers essential topics like the Linux command line, file system navigation, basic shell scripting, and managing files and directories. Students also learn about user management, permissions, and basic system administration tasks. This course equips learners with foundational Linux skills, making it ideal for those new to the platform or preparing for more advanced Linux certifications.

  • Course Duration: Self-paced
  • Cost: Free

Cyber Security

Introduction To Cyber Security.

This course introduces the key principles of cybersecurity, covering the basics of online threats, data protection, and risk management. Students explore topics such as network security, encryption, malware, and secure practices for both individuals and organizations. The course provides a foundational understanding of cybersecurity concepts, preparing learners for more advanced study and helping them recognize and mitigate common cyber threats.

  • Course Duration: Self-paced
  • Cost: Free
